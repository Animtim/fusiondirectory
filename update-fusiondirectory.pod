=head1 NAME

update-fusiondirectory - class cache,locale, and documentation update for FusionDirectory

=head1 SYNOPSIS

update-fusiondirectory [OPTION]

=head1 DESCRIPTION

update-fusiondirectory is a script that help you to manage your fusiondirectory instance

=over 3

=item B<rescan-i18n>       Rebuilds the translations

=item B<rescan-classes>    Rebuilds the class list

=back
